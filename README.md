# mountr

<!-- badges: start -->

<!-- badges: end -->

mountr is a barebones package that allows you to mount remote folders
using `sshfs`.

## Installation

You can install the developement version of `mountr` from GitHub with:

``` r
remotes::install_github("eliocamp/mountr")
```
